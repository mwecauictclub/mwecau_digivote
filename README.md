##  What It Does

- 🗳️ **Multi-level elections** — Presidential, Residential, Course
- 🔐 **Secure voting** — JWT auth, one vote per student, tamper-proof
- 📊 **Real-time results** — instant counting, live analytics
- 🌍 **Vote from anywhere** — students abroad included
- 📱 **Mobile friendly** — works on any device

##  Election Scope

| Type | Who Votes |
|------|-----------|
| Presidential | All MWECAU students |
| Residential | Kifumbu, Mawela, Kwachange, White House, Moshi Mjini, On-Campus |
| Course | All undergraduate, postgraduate & diploma programs |
